APN Enrichment Tool

A first-pass standalone tool for Altagether’s property matching workflow. This version focuses on two operations: fill missing APNs in a sold-properties file using the master data file, then push sold-property facts back into every matching master row by APN. Master matching is deterministic; you can optionally query the LA County public parcel layer for rows still missing an APN. Ambiguous or missing matches are surfaced for review.

Mode 1: Master → Sold
Mode 2: Sold → Master

Inputs

Upload CSV files. The tool will infer likely columns, but you can override them before running a merge.

Column mapping

When set, LA County parcel queries add a 5-digit ZIP match on SitusZIP. Choose “None” to ignore.

Write-back settings

If a named field does not exist in master, the tool will create it in the exported output.

LA County parcel lookup

Master matching is deterministic (normalized address only). LA County lookup uses the Assessor parcel snapshot on ArcGIS; optional ZIP narrows situs queries. Use “LA County lookup only” to re-run parcels on the last sold output (or raw sold) without redoing master. Throttled; only auto-fills when exactly one parcel matches.
